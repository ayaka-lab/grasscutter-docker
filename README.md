# grasscutter-docker
grasscutter的docker版本，快速启动游戏服务器

### 操作模式

1. 安装Docker，DockerCompose

2. 安装Git，如果安装了可以跳过

```bash
sudo apt-get update
sudo apt-get -y install git
```

3. 克隆本仓库
```bash
git clone https://github.com/ayaka-lab/grasscutter-toolkit && cd grasscutter-toolkit
```
